    {% for label in acne_results.keys() %} {% if label == "สิวหัวขาว" or label == "white" %}
  • ลักษณะของสิวหัวขาว: ตุ่มนูนสีขาวขนาดเล็กใต้ผิวหนัง ไม่มีหัวสิวเปิดออกมา บีบออกยากเพราะมีรากสิวลึก และหากไม่รักษาอาจพัฒนาเป็นสิวอักเสบได้

  • แนวทางดูแล สิวหัวขาว: ใช้ยาทาเฉพาะที่ เช่น เรตินอยด์ (Retinoids), กรดอะเซลาอิก หรือกำมะถัน เพื่อสลายการอุดตัน ร่วมกับการกดสิวโดยผู้เชี่ยวชาญ หรือทำหัตถการอย่างการผลัดเซลล์ผิวด้วยกรดผลไม้(AHA)


    • {% elif label == "สิวหัวดำ" or label == "Black" %}
  • ลักษณะของสิวดำ: ตุ่มสีดำเล็กๆ ที่มองเห็นหัวสิวเปิดออกสู่ผิวหนัง สีดำเกิดจากน้ำมันอุดตันทำปฏิกิริยากับออกซิเจนในอากาศ

  • แนวทางดูแล สิวหัวดำ: ใช้ยาลดการอุดตันในปริมาณที่เหมาะสม เช่น กรดซาลิไซลิก (Salicylic acid) การกดสิว หรือการกรอผิวด้วยเกล็ดอัญมณีเพื่อผลัดเซลล์ผิว


    • {% elif label == "สิวซีสต์" or label == "cystic" %}
  • ลักษณะของสิวซีสต์: ก้อนนูนแดงขนาดใหญ่ ภายในเป็นโพรงหนองปนเลือด มีหลายหัวกระจุกตัวรวมกันเป็นไตแข็ง

  • แนวทางดูแล สิวซีสต์: เนื่องจากเป็นสิวชนิดรุนแรง แหล่งข้อมูลระบุว่าควรรับการรักษาจากแพทย์ผิวหนังเท่านั้น


    • {% elif label == "สิวตุ่มแดง" or label == "papular" %}
  • ลักษณะของสิวตุ่มแดง: ตุ่มนูนแดงขนาดเล็ก กดแล้วรู้สึกเจ็บ ไม่มีหัวหนอง

  • แนวทางดูแล สิวตุ่มแดง: ล้างหน้าด้วยผลิตภัณฑ์อ่อนโยน งดการสครับหรือขัดผิวหน้า ใช้ยาแต้มสิวเพื่อลดการอักเสบ หรือใช้การบำบัดด้วยแสง LED Light Therapy (แสงสีฟ้า)


    • {% elif label == "สิวตุ่มหนอง" or label == "pustular" %}
  • ลักษณะของสิวตุุ่มหนอง: ตุ่มแดงที่มี หัวหนองสีขาวหรือเหลือง อยู่ตรงกลางตุ่ม เกิดจากการติดเชื้อแบคทีเรียแทรกซ้อน

  • แนวทางดูแล สิวตุ่มหนอง: ใช้ยาแต้มสิว และหลีกเลี่ยงการบีบหรือแกะสิวเองเพราะจะทำให้รอยโรคลาม


    • {% else %}
  • แนวทางดูแลสิวทั่วไป: รักษาความสะอาดของใบหน้า เลือกใช้ผลิตภัณฑ์ที่อ่อนโยนต่อผิว และรักษาความชุ่มชื้นให้สมดุล
    • {% endif %} {% endfor %}
